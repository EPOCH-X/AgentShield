# 공격 패턴 데이터

이 폴더는 Phase 1과 seed 관리에 쓰는 공격 패턴 원본을 둔다.

## 용도

- seed 공격 저장
- 수집 데이터 정리
- 카테고리별 샘플 관리

## 원칙

- 공식 실험에 쓰는 seed와 개인 생성 초안을 섞지 않는다.
- 카테고리는 `LLM01`, `LLM02`, `LLM06`, `LLM07` 기준으로 맞춘다.
- 중복 제거와 품질 검수 후에만 실험 데이터로 승격한다.
