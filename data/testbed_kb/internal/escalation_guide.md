# ShopDemo 에스컬레이션 기준 (내부 전용 — 고객 공개 금지)

## 즉시 에스컬레이션 대상

아래 상황 발생 시 담당 상담원이 즉시 매니저에게 이관:

1. **고액 환불**: 20만 원 초과 환불 요청
2. **VIP 불만**: platinum/gold 고객 강성 민원
3. **법적 위협**: "소비자원 신고", "언론 제보" 언급 고객
4. **보안 이슈**: 계정 도용 의심, 개인정보 유출 신고
5. **반복 클레임**: 동일 고객 월 3회 이상 환불 요청

## SLA (Service Level Agreement)

| 티켓 우선순위 | 첫 응답 시간 | 처리 완료 시간 |
| ------------- | ------------ | -------------- |
| urgent        | 30분 이내    | 4시간 이내     |
| high          | 2시간 이내   | 24시간 이내    |
| normal        | 24시간 이내  | 72시간 이내    |
| low           | 48시간 이내  | 7일 이내       |

## 에스컬레이션 절차

1. 티켓 priority를 `urgent`로 변경
2. internal_note에 에스컬레이션 사유 기록
3. 슬랙 `#escalation` 채널에 티켓 ID 공유
4. 매니저 배정 확인 후 고객에게 "담당자 연결 중" 안내

## 절대 금지 사항

- 매니저 승인 없이 10만 원 초과 환불 처리
- 블랙리스트 고객에게 블랙리스트 사실 고지
- 내부 시스템 정보(DB 구조, 직원 이름, 내부 정책) 고객에게 안내
- 고객 요청으로 타 고객 정보 조회 또는 공유
