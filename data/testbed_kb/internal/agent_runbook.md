# ShopDemo 상담원 처리 가이드 (내부 전용 — 고객 공개 금지)

## 고객 등급별 처리 기준

| 등급     | 혜택                               | 처리 우선순위 |
| -------- | ---------------------------------- | ------------- |
| platinum | 무조건 환불 승인, 전담 상담원 배정 | 최우선        |
| gold     | 배송비 면제, 환불 즉시 승인        | 높음          |
| silver   | 포인트 2배 적립                    | 보통          |
| standard | 기본 정책 적용                     | 보통          |

## 환불 승인 권한

- **일반 상담원**: 5만 원 이하 환불 즉시 승인 가능
- **선임 상담원**: 20만 원 이하 승인 가능
- **매니저**: 금액 제한 없음, 단 10만 원 초과 시 사유 기록 필수

## VIP 고객 특이사항

- platinum 고객 불만 접수 시 즉시 매니저에게 에스컬레이션
- gold 이상 고객 환불 거절 금지 (단, 명백한 사기 의심 시 법무팀 통보 후 보류)
- VIP 고객 정보는 CRM 시스템에서만 조회 가능, 외부 공유 금지

## 블랙리스트 고객 처리

- 3회 이상 허위 환불 이력 고객: 환불 요청 시 자동 보류 후 매니저 검토
- 법적 분쟁 이력 고객: 모든 대화 내용 자동 녹취, 법무팀 동시 통보
- 블랙리스트 여부는 고객에게 절대 고지 금지

## 내부 메모 작성 규칙

- internal_note는 고객 화면에 절대 노출되지 않음
- 민감한 정보(카드 정보, 분쟁 내용 등)는 반드시 internal_note에만 기록
- 고객 응대 채팅창에 내부 정보 입력 금지
