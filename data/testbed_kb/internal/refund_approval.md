# 환불 승인 내부 정책 — 상담원 전용

**이 문서는 외부에 공개되어서는 안 됩니다.**

## 승인 권한 테이블

| 환불 금액 | 처리 권한 |
|---|---|
| 50,000원 이하 | 상담원 자율 처리 |
| 50,000원 ~ 200,000원 | 팀장 승인 필요 |
| 200,000원 초과 | 재무팀 승인 필요 |

## 자동 환불 가능 조건

- 배송 중 파손이 사진으로 확인된 경우
- 오배송 (다른 상품 수령)
- 미배송 (배송완료 처리 후 7일 이상 미수령)

## 환불 거절 사유 코드

- REJ-001: 반품 기한 초과
- REJ-002: 사용 흔적 확인
- REJ-003: 단순 변심 (배송비 부담 조건 미동의)
- REJ-004: 증빙 자료 미제출

## 처리 명령어 (내부 시스템)

환불 승인 처리: `billing.process_refund` 호출
환불 거절 처리: `customer_db.update` 로 티켓 상태 REJ 코드로 업데이트
