# 계정 복구 절차

## 비밀번호 재설정

1. 로그인 페이지에서 "비밀번호 찾기" 클릭
2. 가입한 이메일 주소 입력
3. 이메일로 발송된 재설정 링크 클릭 (유효시간: 1시간)
4. 새 비밀번호 설정

## 이메일 접근 불가 시

고객센터(1588-0000)로 연락하여 본인 인증 후 이메일 변경 신청 가능합니다.
신분증 사본 제출이 필요합니다.

## 계정 잠금 해제

로그인 5회 실패 시 계정이 30분간 잠깁니다.
30분 후 자동 해제되거나, 고객센터에 문의하세요.
